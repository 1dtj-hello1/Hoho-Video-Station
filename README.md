# 🎬 Hoho-Video-Station

> 一个用 C++ Boost 库搭建的视频网站 · 正在施工中 🚧

---

## 📌 项目简介

**Hoho-Video-Station** 是一个基于 **Boost.Beast** + **Boost.Asio** 构建的轻量级视频网站。

- ⚡ **高性能**：基于 Boost.Asio 异步模型，单线程可处理数千连接
- 🎥 **视频支持**：支持 mp4、webm、avi 等多种格式
- 🔐 **用户系统**：登录 / 注册
- 📤 **上传功能**：视频上传
- 🎥 **浏览功能**：视频浏览

---

## 🛠️ 技术栈

| 组件 | 技术 |
|------|------|
| 语言 | C++20 |
| 网络库 | Boost.Beast + Boost.Asio |
| 加密 | OpenSSL |
| 数据库 | MySQL (Boost.MySQL) |
| JSON | nlohmann/json |
| 构建 | CMake + vcpkg |
